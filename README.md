# Dotfiles

Dotfiles for macos & New Mac Setup

## Usage for syncing .config

```shell
$ git clone https://github.com/gdevtech/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./setupConfig.sh
```

### New Mac Setup
```shell
$ cd ~/.dotfiles
$ cd scripts
$ ./appsInstallation.sh
$ cd ~/.dotfiles
$ ./setupConfig.sh
```
